Wersja Minecraft: `1.21.6`\
Adres: `co-tam-sie-odjaniepawla.hack4krak.pl:25566`

## Wskazówki

- Serwer jest non-premium, więc nie trzeba mieć kupionej gry, żeby na niego wejść. Możesz pobrać i uruchomić ją w ten sposób:
```bash
git clone https://github.com/FabricMC/fabric-example-mod.git && cd fabric-example-mod && git checkout 1.21.6 && ./gradlew runClient
```
